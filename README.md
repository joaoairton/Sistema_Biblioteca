# Sistema_Biblioteca
 Trabalho prático de Programação II utilizando Python

 • O sistema possui duas classes principais: Livro e Biblioteca.

 • Além disso, possui as funcionalidades de adicionar um novo livro à biblioteca, remover, editar, pesquisar por título, exibir todos os livros e permitir o empréstimo e devolução de livros, com controle de disponibilidade.
